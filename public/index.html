Courier Connection Api API Reference

Rest api to request waybills to different delivery services such as ESTAFETA, DHL, FEDEX, REDPACK, PAQUETEEXPRESS.

API Endpoint
https://0.0.0.0:3000/api
Request Content-Types: application/json, application/xml
Response Content-Types: application/json
Schemes: https, http
Version: 1.14.1
